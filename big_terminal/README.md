## 🔧 Установка

1. Поместите папку `big_terminal` в `garrysmod/gamemodes/helix/plugins/`
2. Перезапустите сервер
3. Используйте предметы "Терминал" и "Точка доступа" из категории "Компьютеры"

---

## 🎮 Основные возможности

- **Терминал** — основной интерфейс для всех хакерских операций
- **Точки доступа** — устройства для подключения к сети и взлома
- **Вирусы** — программы для атаки на игроков и точки
- **Антивирусы** — программы для защиты от атак
- **Казино** — система для заработка коинов и открытия свойств
- **Форумы** — система общения между игроками

## 💻 Терминал

### Команды терминала

| Команда | Описание |
|---------|----------|
| `login` | Войти в аккаунт терминала |
| `reg <имя>` | Создать новый аккаунт |
| `user_info` | Показать статистику пользователя |
| `help` | Показать список команд |

### Регистрация

```
reg my_nickname
login
```

После регистрации вы получаете доступ ко всем функциям терминала.

## 🔓 Система взлома

### Взлом игроков

```
hack_user <имя_игрока>
```

**Требования:**
- Подключение к активной точке доступа
- Цель должна быть в радиусе 2000 юнитов от точки
- Цель должна иметь аккаунт терминала

**Процесс:**
1. Открывается мини-игра Cyberpunk (матрица символов)
2. Нужно найти и выбрать правильные последовательности
3. Время на взлом: 60 секунд
4. При успехе цель взломана на 5 минут

**Штраф за провал:**
- Кулдаун 2 часа на повторный взлом этой цели

### Хэш-анализ

```
hash_data_scan
```

Мини-игра для анализа хэш-кодов. Нужно выбрать код с правильной спектрограммой.

## ⚡ Эффекты взлома

После успешного взлома игрока доступны следующие эффекты:

### Базовые эффекты (уровень 0+)

| Команда | Эффект | Кулдаун |
|---------|--------|---------|
| `User_hack_detecting` | Делает цель доступной для взлома на 5 минут | — |
| `User_hack_spycam` | Показывает позицию взгляда цели | 60 сек |
| `User_hack_glitch` | Эффект глича на экране цели (5 сек) | 2 мин |
| `User_hack_WeaponJam` | Заклинивает оружие цели (30 сек) | 3 мин |
| `User_hack_Invert` | Инвертирует управление (10 сек) | 5 мин |

### Продвинутые эффекты (уровень 25+)

| Команда | Эффект | Кулдаун |
|---------|--------|---------|
| `User_hack_Freez` | Замораживает цель на 3 секунды | 2 мин |
| `User_hack_Shok` | Наносит 25 урона электрическим шоком | 4 мин |
| `User_hack_Voicetrap` | Прослушка голоса цели (30 сек) | 2 мин |

### Мастер-ключ

Специальный режим для администратора (SteamID: `STEAM_0:1:217191793`):

```
big_masterkey
```

Позволяет:
- Использовать эффекты без предварительного взлома
- Пропускать мини-игры
- Взламывать без кулдаунов

## 🦠 Вирусы

### Создание вируса

```
virus_create
```

Открывает мини-игру рулетки свойств. Можно выбрать до 3 свойств.

**Кулдаун:** 15 минут

### Управление вирусами

| Команда | Описание |
|---------|----------|
| `virus_list` | Список ваших вирусов |
| `virus_info <номер>` | Информация о вирусе |
| `virus_delete <номер>` | Удалить вирус |
| `virus_rename <номер> <имя>` | Переименовать вирус |
| `virus_share <номер> <игрок>` | Передать вирус (30+ ур) |
| `virus_properties_list` | Список доступных свойств |

### Установка вируса на точку

```
virun_acc_point_instal <номер_вируса>
```

Вирус устанавливается на последнюю отсканированную точку.

### Сканирование точки на вирус

```
virun_acc_point_scan <код_точки>
```

**Время сканирования:** 5 минут

### Обезвреживание вируса

```
virun_acc_point_uninstal <код_точки>
```

Открывает мини-игру для обезвреживания.

## 🛡️ Антивирусы

### Создание антивируса

```
virus_defend
```

**Требование:** 15+ уровень нетраннинга

**Кулдаун:** 15 минут (общий с вирусами)

### Управление антивирусами

| Команда | Описание |
|---------|----------|
| `antivirus_list` | Список ваших антивирусов |
| `antivirus_delete <номер>` | Удалить антивирус |
| `antivirus_properties_list` | Список доступных свойств |

### Установка антивируса

```
access_point_owner_key <код_точки>
```

Получить мастер-ключ для своей точки (действителен 5 минут).

```
antivirus_install <мастер-ключ> <номер_антивируса>
```

### Проверка антивируса

```
antivirus_check <мастер-ключ>
```

## 📡 Точки доступа

### Подключение к точке

```
access_point <код>
```

Код — 6-значный номер точки. Действителен 30-60 минут.

### Управление точками

| Команда | Описание |
|---------|----------|
| `access_point_list` | Список подключённых точек |
| `access_point_delete <код>` | Удалить точку из списка |
| `access_point_ping` | Показать игроков рядом с точками |
| `access_point_users <код>` | Кто подключён к точке |
| `access_point_find_local <код>` | Найти точки рядом |

### Уничтожение точки

```
access_point_destroy <код>
```

- Свою точку — уничтожается сразу
- Чужую точку — требуется 25+ уровень и капча

### Взлом точки

```
access_point_hack <код>
```

**Требование:** 25+ уровень нетраннинга

Сложная мини-игра с повышенной сложностью.

## 🎰 Казино

### Открытие казино

```
casino
```

### Заработок коинов

#### Кликер

Нажимайте на кнопку для заработка коинов.

- **Базовая сила клика:** 0.1 коина
- **Улучшение:** +50% за уровень

#### Пассивный доход

- **Базовый доход:** 0.01 коина/сек за уровень
- **Сбор:** `Собрать пассивный доход`

### Апгрейды

| Апгрейд | Формула стоимости |
|---------|-------------------|
| Кнопка | 10 × 1.5^уровень |
| Пассивный доход | 50 × 2^уровень |

### Игры

#### Crash (Ракетка)

- Ставка: любая сумма коинов
- Множитель растёт с ускорением
- Нужно успеть забрать до краша
- **Максимальный множитель:** x10
- **Риск:** Мгновенный краш при 1.0x

#### Рулетка

| Ставка | Выигрыш |
|--------|---------|
| Красное/Чёрное | x2 |
| Половина (1-18 / 19-36) | x2 |
| Дюжина (1-12 / 13-24 / 25-36) | x3 |
| Конкретное число | x36 |

**Номера:** 0-36 (0 = зелёный)

#### Слот-машина

**Символы:**
- `7777` — Джекпот (x10)
- `GEMS` — Драгоценности (x8)
- `STAR` — Звезда (x6)
- `BELL` — Колокол (x5)
- `BARS` — Бары (x4)
- `CHRY` — Вишня (x3)
- `LMON` — Лимон (x3)

**Комбинации:**

| Комбинация | Шанс | Множитель |
|------------|------|-----------|
| 7777 x3 | Очень редкий | x10 |
| GEMS x3 | Редкий | x8 |
| STAR x3 | Редкий | x6 |
| BELL x3 | Редкий | x5 |
| BARS x3 | Редкий | x4 |
| Другие x3 | Редкий | x3 |
| x2 одинаковых | 10% | x2 |
| Вишня x1 | 20% | x1.5 |

### Рулетка свойств

**Стоимость:**

## 🔒 Защита фракций

### Защищённые фракции

Следующие фракции **невозможно взломать**:

- Администратор сервера
- admin
- superadmin
- operator

Также защищены:
- Суперадминистраторы (IsSuperAdmin)
- Игроки с мастер-ключом

### Наказание за попытку взлома

При попытке взломать защищённую фракцию:

```
═══════════════════════════════════════
  [КРИТИЧЕСКАЯ ОШИБКА]
═══════════════════════════════════════

  Я очень сухой мальчик…
  Я хочу пить…

  [СИСТЕМА] ВЗЛОМ ЗАПРЕЩЁН
  [СИСТЕМА] ПРИМЕНЕНО НАКАЗАНИЕ
═══════════════════════════════════════
```

**Наказание:**
- Голод = 0
- Жажда = 0
- Здоровье = 50% от текущего

## 💾 Сохранение данных

### Автосохранение

- При создании/удалении терминала
- При создании/удалении точки доступа
- При изменении данных казино
- При создании/удалении вирусов/антивирусов

### Сохранение при выключении

Все данные сохраняются автоматически при:
- `ShutDown` — выключение сервера
- `ChangeLevel` — смена карты

### Хранение данных

- `ix.data` — терминалы, точки, форумы, пользователи
- `ixhl2rp/big_terminal/casino_data.txt` — данные казино
